## Quick Start

1. Clone the repository
```bash
git clone https://github.com/Shisaku918/Portfolio-template.git
```

2. Add your images to the `img/` folder

3. Edit `index.html`:
   - Replace "Your name/pseudo" with your name
   - Update the about text
   - Add your projects
   - Change contact email

4. Open `index.html` in your browser

## Customization

### Colors
Main colors in `style.css`:
- `deeppink` - Pink glow
- `blueviolet` - Purple glow
- `indigo` - Blue glow

### Font
The site uses **Poppins** from Google Fonts. Change it in the `<head>` section.

## Responsive Breakpoints

- **Desktop**: > 1024px
- **Tablet**: 768px - 1024px (hamburger menu)
- **Mobile**: < 768px (vertical layout)
- **Small mobile**: < 480px (hidden profile picture)
